变量

变量使用的注意事项：

  1. 名字不能重复
  2. 变量未赋值，不能使用
  3. long类型的变量定义的时候，为了防止整数过大，后面要加l或L
  4. float类型的变量定义的时候，为了防止类型不兼容，后面要加f或F
