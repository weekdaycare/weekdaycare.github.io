轻量云服务期搭建Ondrive离线下载

2022-06-28- 2022-07-04
919-
捣鼓-技术

本文使用的是腾讯云的轻量应用服务器

废话不多说直接开始，或者看这个贴子

LightUploader，一个轻量化的网盘上传工具，支持OneDrive国际/个人/家庭版、世纪互联、Google Drive – 高玩梁的博客 (gwliang.com)

系统

轻量默认配了宝塔面板

重装时很快，不到一分钟就好了
重装选择宝塔腾讯云版，直接省去装宝塔面板过程

ARIA2

直接使用github大佬的aria2增强脚本，提速超级明显

wget -N http://git.io/aria2.sh && chmod +x aria2.sh

若上面的代码无法下载
则使用下面的代码

wget -N --no-check-certificate https://raw.githubusercontent.com/P3TERX/aria2.sh/master/aria2.sh && chmod +x aria2.sh && bash aria2.sh

完成之后会看到如下代码

按1安装Aria2
安装完成后输入./aria2.sh再次进入该界面输入8查看配置如下

记住RPC端口，在服务器防火墙中将改端口设为允许
复制AriaNg链接并在浏览器中打开

上传

最近有个大佬写了一款新的工具
LightUploader
简单粗暴可以抛弃以前的rclone了

下载文件

wget https://github.com/gaowanliang/LightUploader/releases/download/v2.0/LightUploader_Linux_x86_64.tar.gz

解压文件

tar -zxvf LightUploader_Linux_x86_64.tar.gz -C /usr/local/bin/

给予权限

chmod +x /usr/local/bin/LightUploader

可选 软链接 类似于 Windows 的快捷方式

ln -s /usr/local/bin/LightUploader /usr/bin/LightUploader

安装完成后

掏出我那限制的onedrive E5

登录

登录到您的帐户 (microsoftonline.com)

地址栏会出现一个localhost开头的链接，复制

在ssh上输入

LightUploader -a "url" -l zh-CN

将url部分替换为你的链接

如果提示：注册成功 ok

配置文件

这时候宝塔面板就派上用场了

在/root/.aria2c下有一个aria2.conf

找到on-download-complete=

改成

on-download-complete=/root/.aria2c/upload.sh

打开upload.sh文件

改成

#!/bin/bash

GID="$1";
FileNum="$2";
File="$3";
MaxSize="157286400";
Thread="3";  #默认3线程，自行修改，服务器配置不好的话，不建议太多
Block="20";  #默认分块20m，自行修改
RemoteDIR="";  #上传到Onedrive的路径，默认为根目录，如果要上传到Test目录，""里面请填成Test
LocalDIR="/www/download/";  #Aria2下载目录，记得最后面加上/
Uploader="/usr/local/bin/LightUploader";  #上传的程序完整路径，默认为本文安装的目录
Config="/root/auth.json";  #初始化生成的配置auth.json绝对路径，参考第3步骤生成的路径


if [[ -z $(echo "$FileNum" |grep -o '[0-9]*' |head -n1) ]]; then FileNum='0'; fi
if [[ "$FileNum" -le '0' ]]; then
	exit 0; fi
if [[ "$#" != '3' ]]; then
	exit 0; fi

if [[ ! -e "${Uploader}" ]]; then
	exit 0; fi
IFS_BAK=$IFS
IFS=$'\n'
tmpFile="$(echo "${File/#$LocalDIR}" |cut -f1 -d'/')"
FileLoad="${LocalDIR}${tmpFile}"
if [[ ! -e "${FileLoad}" ]]; then
	exit 0; fi
ItemSize=$(du -s "${FileLoad}" |cut -f1 |grep -o '[0-9]*' |head -n1)
if [[ -z "$ItemSize" ]]; then
	exit 0; fi
if [[ "$ItemSize" -ge "$MaxSize" ]]; then
	echo -ne "\033[33m${FileLoad} \033[0mtoo large to spik.\n";
	exit 0; fi
${Uploader} -c "${Config}" -t "${Thread}" -b "${Block}" -f "${FileLoad}" -r "${RemoteDIR}"
if [[ $? == '0' ]]; then
	rm -rf "${FileLoad}";
fi
IFS=$IFS_BAK

保存就完事了

这时候打开之前的AriaNg链接 左上角new添加任务

过几分钟，你下载的东西就能在onedrive上查看了
